# IFRIT Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- CLI manager tool for database operations
- Pattern management commands
- Attacker profile viewing and management
- Exception/whitelist management via CLI
- Comprehensive attack filtering and analysis

### Changed
- Improved database schema with additional indexes
- Enhanced anonymization engine performance

### Fixed
- Database connection pooling issues
- Honeypot response timing accuracy

## [1.0.0] - 2025-11-02

### Added
- Initial release of IFRIT Proxy
- Reverse proxy with HTTPS/TLS support
- Real-time threat detection with Claude and GPT integration
- SQLite-based self-learning attack patterns
- Data anonymization before external API calls
- Read-only web dashboard with token authentication
- REST API for integrations
- Full YAML configuration support
- Docker and systemd deployment options
- Attacker profiling system
- Honeypot payload generation
